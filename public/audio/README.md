# Audio assets

Drop your own MP3s into this folder. The runtime falls back to synthesised placeholders if a file is missing.

Recognised filenames:

| File | When it plays |
| --- | --- |
| `flip.mp3` | A card is flipped (single or stack) |
| `pickup.mp3` | A card is grabbed |
| `place.mp3` | A card is dropped |
| `shuffle.mp3` | A stack is shuffled (Ctrl + scroll down) |
| `gather.mp3` | A stack is gathered (Ctrl + scroll up) |
| `snap.mp3` | A card snaps back from a rival zone |
| `ui-click.mp3` | A generic UI button click |
| `ui-open.mp3` | A modal opens |
| `ui-close.mp3` | A modal closes |
| `music.mp3` | Background loop (auto-loops on play) |

Keep individual SFX shorter than 250 ms for snappy feel. `music.mp3` is looped indefinitely.

Volumes can be adjusted in-game from the Settings panel.
